# VIVUM 2026 — The Ascent

The official website for **Vivum 2026**, the annual inter-school festival of
The International School Bangalore (TISB).

A scroll-driven 3D expedition up a mountain — from base camp to the summit —
where each checkpoint opens into the festival's sports and cultural events.

🌐 **Live:** https://vivum.tisb.org

## About

- **13 & 14 August 2026** · TISB, Bengaluru
- Sports, cultural events, and registration details for all participating schools
- A lightweight, reduced-motion version is available for accessibility and slower devices

## Built with

- React + TypeScript
- Three.js / React Three Fiber (the 3D world)
- Vite
